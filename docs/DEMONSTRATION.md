# Демонстрация проекта

## Что делает приложение

1. **Хранит информацию об играх**:
   - Название
   - Платформа
   - Количество игроков
   - Статус (пройдена/не пройдена)

2. **Основные функции**:
   - Добавление новых игр
   - Просмотр всей коллекции
   - Удаление игр
   - Автосохранение в файл

## Как показать работу

### Сценарий демонстрации (2-3 минуты):

1. **Запуск приложения**:
2. **Показ загруженных игр** (если есть):
- Выберите пункт 2 в меню

3. **Добавление новой игры**:
- Выберите пункт 1
- Введите тестовые данные:
  - Название: "It Takes Two"
  - Платформа: "PC"
  - Игроков: "2"
  - Пройдена: "да"

4. **Просмотр добавленной игры**:
- Снова выберите пункт 2

5. **Сохранение данных**:
- Покажите, что данные сохраняются автоматически
- Можно выйти и зайти снова - игра останется в списке

6. **Завершение работы**:
- Выберите пункт 5 для выхода
## Преимущества приложения

- Простой и понятный интерфейс
- Данные не теряются после закрытия
- Не требует установки дополнительных программ
- Код хорошо структурирован
- Есть тесты
